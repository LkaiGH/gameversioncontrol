说明：

中国科学院大学2015年硕士研究生入学考试业务课统一命题科目考试大纲，仅供大家在复习备考中参考。中国科学院大学各研究所、各院系的业务课用题途径（①国科大统一命题，②各研究所各院系自命题，③其它用题方式）是由各研究所和各院系自由确定的，具体请咨询你准备报考的研究所或院系。

2015年硕士入学考试"计算机科学与技术"一级学科第四单元考试科目为国家统一命题的 "计算机学科专业基础综合"（考试科目代码408）或国科大统一命题的"计算机学科综合（专业）" （考试科目代码863），具体选用考试科目请参见各研究所和各院系的2015年硕士招生专业目录。中国科学院大学统一命题的"计算机学科综合(非专业)"（考试科目代码862）主要针对"计算机科学与技术"一级学科以外的需要考核计算机业务课的其他专业。

硕士研究生入学考试是选拔性考试，要求考生对本学科专业的基础理论、基本知识和基本技能灵活掌握，同时具有运用理论解决实际问题的能力。请同学们复习备考时要宽泛阅读，灵活掌握，而不要死记硬背。

另外，除提供2012年真题（http://admission.gucas.ac.cn/home/detail/6fe3e475-f08f-44ce-96ac-b1e19f06d87d），以及2013年真题（http://admission.ucas.ac.cn/home/detail/7fd78328-723a-464e-b36f-d7db4f3ac149）外不提供其他年份的试题，且真题仅供参考。

如有其他疑问，请联系中国科学院大学招生办公室，电话：010-88256215或82640446。

祝各位考生考试成功！
